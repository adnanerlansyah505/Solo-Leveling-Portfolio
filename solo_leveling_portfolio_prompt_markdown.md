# Solo Leveling Inspired Developer Portfolio Website

## Overview

Create a cinematic, anime-inspired developer portfolio website using **NuxtJS 4**, **TailwindCSS**, and modern animation libraries.

The UI/UX aesthetic should feel heavily inspired by dark fantasy RPG systems and "Solo Leveling"-style holographic interfaces, featuring glowing purple neon energy, magical UI transitions, animated particles, smooth hover interactions, and futuristic dashboard panels.

The website must feel:

- Premium
- Immersive
- Futuristic
- Highly animated
- Responsive
- Interactive
